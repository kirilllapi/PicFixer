Метод Собеля

Метод Собеля - это один из методов извлечения краев из изображений. Операция Собеля состоит из двух разных ядер. Один для поиска горизонтальных краев, а другой для поиска вертикальных краев. Затем результаты двух ядер объединяются, чтобы обнажить все значимые края изображения.
